# runelitemultilauncher
a python based launcher that launches multiple instances of runelite at once for convenient multi logging, has many other tools
copy and paste all Launcher.zip files into the Runelite directory(Hayden.zip is NOT needed) (right click Runelite shortcut on Desktop and click proporties then click open file location)
create a desktop shortcut to Launcher.py for convienence
launch Lanncher.py

Enjoy!
